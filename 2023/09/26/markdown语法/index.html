Markdown语法

(1)标题

在文字前加 # ,有1-6级标题

效果如下：

也可以在文字底下加 = 显示一级标题，加 - 显示二级标题

效果如下：

(2)字体

* 加粗,倾斜,斜体加粗
~ 号删除线,下标
^ 号上标
$设置数学表达式
注意使用英文符

加粗
倾斜
斜体加粗
删除线
我是^上标^
我是下标
$$int_1^1$$

(3)列表

+ 或者 num + . + space 生成不同样式列表

  • 一二三四五
    • 上山打老虎
      • 老虎不在家
        • 打到小松鼠
  1. 一二三四五
  2. 上山打老虎
  3. 老虎不在家
  4. 打到小松鼠

(4)表格

第一行是标题行,表格不识别空格
第二行是分割行,可以选择该列居中格式(感觉表格还是直接生成比较方便)

MON TUE WED THU FRI
chr chr chr
xx xx xx

(5)引用

使用 > 生成引用,有多级引用

一二三四五

上山打老虎

老虎不在家

打到小松鼠

松鼠有几只
快来数一数

一二三四五

(6)分割线

用三个以上 - 或 * 或 _ 生成分割线
要空一行,不然样式不同

(7)代码

用不同个数的 ` 生成单行代码和多行代码

例如：

`我是代码`

我是代码

(8)链接语法

这是一个链接链接文字

https://www.bing.com
1713442739@qq.com

https://www.bing.com

(9)图片

插入图片markdown语法 ![图片alt](图片链接 "图片title") 感觉直接图床复制md链接粘贴比较实在（图床如何配置之后有空再写篇Blog，也可以询问tg）

例如：

![](https://cdn.jsdelivr.net/gh/Ming3z/Ming3_Img/img/markdown_test001.jpeg)

(10)转义字符

以下列出的字符都可以通过使用反斜杠 \ 从而达到转义目的

Character Name
\ backslash
` backtick
* asterisk
_ underscore
{ } curly braces
[ ] brackets
( ) parentheses
# pound sign
+ plus sign
